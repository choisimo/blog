구간 DP — Matrix Chain Multiplication

📌 알고리즘

Interval DP (최적 분할)

💡 입력

dims=[10, 30, 5, 60]
(A: 10×30, B: 30×5, C: 5×60) 의 행렬 3개

⏱ 복잡도

Time O(n³)Space O(n²)

🎓 학습 포인트

  • 배열의 인덱스와 실제 행렬의 차원이 어떻게 매핑되는지 (dims의 활용)
  • 점화식에서 두 집합을 결합하는 과정과, 결합할 때 발생하는 dims[i]*dims[k+1]*dims[j+1] 연산 비용의 원리
  • 분할 지점 k를 변경해 가며 최적의 cost를 구하는 과정 추론
